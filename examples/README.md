Examples
========

Current layout uses one subfolder per IDE/toolchain. At the moment only STM32CubeIDE projects are provided under:

- `examples/stm32cubeide/` (e.g., Nucleo-575ZI-Q/03_TimerISRJitter)

Future IDEs (VS Code, Keil, IAR, etc.) should add their own top-level subfolder under `examples/` following the same pattern.
